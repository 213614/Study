Grid

● Bootstrap Grid System

- CSS에 media query를 bootstrap이 아예 시스템화해놓음
→ 가로 한 줄을 12칸으로 쪼개놓고 알아서 개발자가 크기 조절해서 사용하면 됨 (테이블처럼)
- 12개의 칼럼을 지원하는 시스템
- 한줄(row)은 총 12칸으로 구성되어 있다

(참조 : https://www.w3schools.com/bootstrap/bootstrap_grid_basic.asp )

● [칼럼의 클래스 종류]

.col-xs-* 항상 가로로 배치 (phones) *모바일용(무조건 그리드가 가로 한 줄로만 구성됨)
.col-sm-* 768px 이하에서 세로로 표시 시작(tablets) *많이 사용함
.col-md-* 992px 이하에서 세로로 표시 시작(small laptops) *많이 사용함
.col-lg-* 1200px 이하에서 세로로 표시 시작(laptops and desktops)
